# ABCD
ブロック単位でのコードクローン（重複コード）を検出するツールです．<br>
Javaのソースコードに対して実行することができます．<br>
現在，タイプ1とタイプ2のクローンに加え，Extract Method リファクタリングを行いやすいようなクローンのみを検出することができます．
